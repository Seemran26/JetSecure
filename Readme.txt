Reservation Site

Introduction: 

The Reservation Site is a web application that allows users to manage their reservations and view their flight data. The application supports user login and sign-up functionalities. Users can view, edit, and delete their data, like personal details.

Features

- User Registration and Login: Users can sign up and log in to access their account.
- View Data: After logging in, users can view their personal login details.
- Edit Data: Users can edit their personal information.
- Delete Data: Users can delete their accounts if needed.

Technologies Used

- Frontend: HTML, CSS, Bootstrap
- Backend: PHP
- Database: MySQL
- Server: Apache (XAMPP)
